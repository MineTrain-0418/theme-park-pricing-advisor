# テーマパーク価格戦略シミュレーター

テーマパークの経営パラメータを入力すると、Claude AIが収益最大化の観点から価格戦略アドバイスを返すCLIツールです。

## 機能

- 対話形式で6項目を入力するだけでアドバイスを生成
- Claude API（ストリーミング）によるリアルタイム出力
- `--mock` フラグでAPIキーなしの動作確認が可能

## セットアップ

**必要環境:** Python 3.10以上

```bash
pip install -r requirements.txt
```

APIキーを環境変数に設定してください：

```bash
# Windows (コマンドプロンプト)
set ANTHROPIC_API_KEY=your-api-key-here

# Windows (PowerShell)
$env:ANTHROPIC_API_KEY = "your-api-key-here"

# Mac/Linux
export ANTHROPIC_API_KEY=your-api-key-here
```

> APIキーは [Anthropic Console](https://console.anthropic.com/) で取得できます。

## 使い方

### 通常実行（API使用）

```bash
python main.py
```

### モックモード（APIキー不要・動作確認用）

```bash
python main.py --mock
```

### 実行例

```
============================================================
  テーマパーク価格戦略シミュレーター
============================================================
以下の項目を順に入力してください。

1. パーク種別を選んでください (ディズニー系 / USJ系 / その他): ディズニー系
2. 現在のチケット価格（円）を入力してください: 10000
3. 想定混雑度を1〜10で入力してください (1=空いている, 10=超混雑): 7
4. 年間来場者数（万人）を入力してください: 3000
5. 時期を選んでください (繁忙期 / 閑散期): 繁忙期
6. ダイナミックプライシング導入済みですか？ (はい / いいえ): いいえ

分析中です。しばらくお待ちください...
```

## 入力項目

| # | 項目 | 選択肢 / 形式 |
|---|------|--------------|
| 1 | パーク種別 | `ディズニー系` / `USJ系` / `その他` |
| 2 | チケット価格 | 正の整数（円） |
| 3 | 混雑度 | `1`（空いている）〜 `10`（超混雑） |
| 4 | 年間来場者数 | 正の数（万人） |
| 5 | 時期 | `繁忙期` / `閑散期` |
| 6 | ダイナミックプライシング | `はい` / `いいえ` |

## 出力内容

Claude が以下の4項目を日本語でアドバイスします：

1. **現状の収益ポジション評価** — 業界水準・競合との比較分析
2. **価格最適化の提案** — 具体的な価格施策と価格帯の目安
3. **リスクとトレードオフの分析** — 来場者数・ブランドへの影響
4. **経営者向け一言サマリー** — 最重要アクションの要約
